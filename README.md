<h1 align="center">Welcome to web-store</h1>

> Build a web application have authenticate, add, edit product from user, order and checkout cart.

## Prerequisites

* node >= 12.14.1
* npm >= 6.13.4

## Install

```sh
npm install
```

Edit the content in .env file with your mongodb uri

## Usage

```sh
npm start
```
